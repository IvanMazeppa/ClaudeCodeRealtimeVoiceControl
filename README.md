# Voice Coding Research

This repository is an isolated workspace for voice-coding research and workflow hardening.

## Tracks

### `apps/claude_code_voice`

Stable Claude Code plus voice-mode integration work. This track is for preserving and improving the current workflow, prompts, docs, and version-pinned patch materials.

### `apps/realtime_voice`

Future experimental realtime voice app work. This track remains placeholder-only until the research direction is approved and implementation details are ready to be written down.

## Repository Layout

- `apps/claude_code_voice/`: stable voice workflow materials
- `apps/realtime_voice/`: future experimental app materials
- `docs/`: shared architecture, workflow, migration, roadmap, and troubleshooting docs
- `templates/`: reusable templates for future tasks
- `scripts/`: helper automation for later tasks
- `patches/voice-mode/8.5.1/`: version-pinned voice-mode patch assets

## Current Status

Task 1 sets up the repository scaffold only. The approved design and bootstrap plan remain under `docs/superpowers/` and are preserved as source material for later tasks.
